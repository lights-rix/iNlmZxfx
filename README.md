## 前言

欢迎来到基于小程序的老孙电子点菜系统开发设计与实现项目的Gitee页面。本项目是一款结合了SSM框架与微信小程序的餐厅点菜系统，旨在为用户提供便捷、高效的点餐体验。以下是本项目的详细介绍。

## 内容介绍

本项目主要针对餐饮行业开发，通过微信小程序实现顾客自助点餐、下单、支付等功能。系统后端采用Java语言，结合Spring、SpringMVC和MyBatis框架进行开发，前端使用JS、Vue、CSS3和Uniapp技术，实现了一套功能完善、易用性强的电子点菜系统。

## 技术介绍

- 语言：Java
- 使用框架：Spring、SpringMVC、MyBatis，微信小程序
- 前端技术：JS、Vue、CSS3，Uniapp
- 开发工具：IDEA/Eclipse，Uniapp
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven: apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16

## 核心代码

以下是项目中一个简单的Java方法，用于查询菜谱列表：

```java
// 菜谱服务类
@Service
public class DishService {

    @Autowired
    private DishMapper dishMapper;

    // 查询菜谱列表
    public List<Dish> getDishList() {
        return dishMapper.selectDishList();
    }
}
```
